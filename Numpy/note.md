### üîπ A. View ‚Äî `arr.view()`

NumPy array-‡¶è‡¶∞ ‡¶â‡¶™‡¶∞ `for` ‡¶≤‡ßÅ‡¶™ ‡¶ö‡¶æ‡¶≤‡¶ø‡ßü‡ßá ‡¶∏‡¶π‡¶ú‡ßá‡¶á ‡¶™‡ßç‡¶∞‡¶§‡¶ø‡¶ü‡¶ø ‡¶â‡¶™‡¶æ‡¶¶‡¶æ‡¶®‡ßá ‡¶ï‡¶æ‡¶ú ‡¶ï‡¶∞‡¶æ ‡¶Ø‡¶æ‡ßü‡•§

### üîπ A. 1D Array Iteration

```python

    print(x)
```


### üîπ B. 2D Array Iteration (Row by Row)

```python

                [4, 5, 6]])

    print(row)
```

```

[4 5 6]
```

### üîπ C. Nested Loop (Element by Element)

```python

    for item in row:
        print(item, end=' ')
```

**Output:**

```

```

```python
                [30, 40],
```

# NumPy ndarray Manipulation (‡¶¨‡¶æ‡¶Ç‡¶≤‡¶æ‡¶Ø‡¶º ‡¶â‡¶¶‡¶æ‡¶π‡¶∞‡¶£‡¶∏‡¶π)

  ‡¶Ø‡ßá‡¶Æ‡¶®: 6 elements ‡¶•‡¶æ‡¶ï‡¶≤‡ßá `(2, 3)` ‡¶¨‡¶æ `(3, 2)` ‡¶ï‡¶∞‡¶æ ‡¶Ø‡¶æ‡¶¨‡ßá, ‡¶ï‡¶ø‡¶®‡ßç‡¶§‡ßÅ `(2, 4)` ‡¶®‡ßü‡•§
